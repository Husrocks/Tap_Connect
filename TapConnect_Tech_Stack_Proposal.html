TapConnect

Technology Stack Proposal v1.0

1. Project Vision

TapConnect is a proximity-based AI networking platform designed to eliminate social friction through instant NFC/BLE contact exchange and intelligent context-aware matching.

2. Frontend (The "Core" Experience)

PlatformNative Android
LanguageKotlin 2.0+
UI FrameworkJetpack Compose
Local StorageRoom DB + DataStore
Why Native Kotlin? Hardware reliability is critical. Native access to the Android Bluetooth and NFC stacks ensures the "Tap" experience is flawless and lag-free, which is impossible to guarantee with cross-platform wrappers.

3. Backend (The "AI" Engine)

API FrameworkFastAPI (Python)
DatabasePostgreSQL (via Supabase)
Real-time SyncSupabase Realtime / WebSockets
AuthenticationSupabase Auth (Social + OTP)
Why FastAPI? As an AI-first app, Python is the primary ecosystem. FastAPI allows us to integrate Gemini LLMs and vector search natively while maintaining lightning-fast performance for proximity data sync.

4. AI & Intelligence Layer

Primary ModelGemini 1.5 Pro
OrchestrationLangChain / LangGraph
Vector Searchpgvector (Semantic Matching)

The AI layer will handle Smart Profile Summaries, Icebreaker Generation, and Priority Ranking of nearby users based on shared interests.

5. Summary Comparison

Compared to a standard web-app stack, this combination focuses on Hardware Reliability and AI Extensibility, ensuring TapConnect can scale from a simple "tap" app into a full-scale professional networking ecosystem.
